Установка

Реализуется с помощью целей сборки модуля из make-целей Установка модуля в БД.  Для корректной установки должна быть установлена актуальная версия OMS ( см.  Версия OMS-файлов).

Параметры установки

INSTALL_VERSIONустанавливаемая версия модуля ( Last при первоначальной установке или номер при обновлении)
LOAD_OPERATORIDлогин и пароль оператора, выполняющего установку ( формат: login/password)
LOAD_USERIDосновной пользователь для установки в БД ( формат: username/password@dbName)

Первоначальная установка

Предварительные требования

  • выдача прав на модуль Subversion пользователю search;
  • выдача прав на модуль Option пользователю search;

Последовательность установки

  • выдача системных привилегий основной схеме, где расположен модуль ( на примере search@???);
make grant LOAD_USERID=sys/???@??? TO_USERNAME=search GRANT_SCRIPT=sys-privs LOAD_OPERATORID=???/???
  • установка основной части модуля ( на примере search@???);
make install LOAD_USERID=search/???@??? INSTALL_VERSION=Last LOAD_OPERATORID=???/???
  • выдача прав для возможности создания батчей основной схеме БД ( где установлен Scheduler);
make grant LOAD_USERID=???/???@??? GRANT_SCRIPT=batch TO_USERNAME=??? LOAD_OPERATORID=???/???
  • установка батчей модуля в основную схему БД;
make install LOAD_USERID2=???/???@??? INSTALL_VERSION=Last LOAD_OPERATORID=???/???
  • настройка опций батчей;
  • установка ролей;
make install LOAD_USERID3=???/???@??? INSTALL_VERSION=Last LOAD_OPERATORID=???/???
  • настройка опций ( pkg_SvnSearcherBase::Наименования опций для соединения с репозиторием , pkg_SvnSearcherBase::Наименования опций для настройки индексирования );
